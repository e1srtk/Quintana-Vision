# Quintana-Vision
Web app for futures and fixed-income traders that combines a structured trade journal, performance analytics, and an AI research copilot for reviewing execution and exploring relative-value ideas.
# Core features
Dashboard: PnL, win rate, expectancy, drawdown, top strategies
Trades: create/edit/filter trades, notes, tags, ideas  
Analytics: strategy breakdowns, instrument breakdowns, arbitrage finder 
AI Copilot: review my last trades, compare setups, analyze metrics 
Research: maintain arbitrage/spread ideas with thesis, risks, and status
