# Quintana-Vision
Web app for futures and fixed-income traders that combines a structured trade journal, performance analytics, and an AI research copilot for reviewing execution and exploring relative-value ideas.
# Core features
1. Dashboard: PnL, win rate, expectancy, drawdown, top strategies, news 
2. Trades: create/edit/filter trades, notes, tags, ideas  
3. Analytics: statistical research, β and α strategy metrics, sharpe ratio 
5. AI Copilot: review my last trades, compare setups, analyze metrics 
6. Research: maintain arbitrage/spread ideas with thesis, risks, and status
